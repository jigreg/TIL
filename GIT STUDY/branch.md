1. 브랜치 생성

  ```bash
  (master) $ git branch {branch name}
  ```

2. 브랜치 이동

  ```bash
  (master) $ git checkout  {branch name} # 구버전
  (master) $ git switch {branch name} # 신버전
  ```

3. 브랜치 생성 및 이동

  ```bash
  (master) $ git checkout -b {branch name} # 구버전
  ```

4. 브랜치 목록

  ```bash
  (master) $ git branch 
  ```

5. 브랜치 병합

   ```bash
   (master) $ git merge {branch name}
   ```

6. 브랜치 삭제

   ```bash
   (master) $ git branch -d {branch name}
   ```
