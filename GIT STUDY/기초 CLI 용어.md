## 기초 CLI 용어

> CLI : Command Lind Interface
>
>   
>
> GUI 환경인 윈도우와는 다른 인터페이스를 갖는다.

- ls : 목록 `ls -l`

  ```bash
  $ ls
  ```

- touch : 파일 만들기

- ```bash
  $ touch <파일명>
  $ touch README.md
  ```

- pwd : 현재 작업 디렉토리 출력 `print working directory`

- mkdir : 디렉토리 만들기 `make directory`

  ```bash
  $ mkdir test
  ```

  

-  cd : 디렉토리 이동 `change directory`

  - `..` : 상위 디렉토리
  - `.`  : 현재 디렉토리
